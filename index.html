Mincer

The poor man's streaming app for macOS

Download 0.2.1 Download 0.0.0 View on GitHub

Mincer is a simple RTMP broadcast application for macOS. It can be used to stream to public live broadcasting sites like Twitch, Hitbox, Ustream or YouTube. Mincer is released under the GNU General Public License.

Mincer is maintained by Florian Zwoch. Tactile theme by Jason Long.
